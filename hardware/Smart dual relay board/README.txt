<p>Simple relay board for quick prototyping of project that need to switch main voltage.</p>
<p> </p>
<p>The board has:</p>
<p>- 38 ESP32 Devkit board</p>
<p>- two relays</p>
<p>- 3 connectors for external switches</p>
<p>- 4 3.3V I2C headers</p>
<p> </p>            
How to use：

At editor, open the document via: Top menu - File - Open - EasyEDA... , and select the json file, then open it at the editor, you can save it into a project.


如何使用：

打开编辑器，通过：顶部菜单 - 文件 - 打开 - 立创EDA... ，选择 json 文件打开在编辑器，你可以保存文档进工程里面。
